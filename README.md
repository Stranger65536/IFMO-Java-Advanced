Решения к курсу «Технологии Java»
====

[Условия домашних заданий](http://www.kgeorgiy.info/courses/java-advanced/homeworks.html)

Сборка
----

* Linux:

        chmod 555 build.sh
        ./build.sh

* Windows:

        build.bat
             
Запуск тестов
----

* Linux:

        chmod 555 runtests.sh
        chmod -R 775 lib/
        ./runtests.sh

* Windows:

        runtests.bat
